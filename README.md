# GoWhere-Crawler
Crawl data for GoWhere project from http://piao.qunar.com/touch/
## 使用
```javascript
//  复制到本地
git clone git@github.com:Jacleklm/Image-Crawler.git

//  定位到该文件夹
cd GoWhere-Crawler

//  安装依赖
npm install puppeteer --save

//  定位到src文件夹
cd src

// 运行文件即可对去哪儿网移动端页面的数据进行爬取
node Crawler.js

```
