# GoWhere-Crawler
Crawl data for GoWhere project from http://piao.qunar.com/touch/
## 使用
```javascript
//  复制到本地
git clone git@github.com:Jacleklm/Image-Crawler.git

//  定位到该文件夹
cd GoWhere-Crawler

//  安装依赖
npm install puppeteer --save

//  新建文件夹（用来放爬取的图片，被我gitignore了）
mkdir ImageCrawler

//  定位到该文件夹
cd src

// 运行文件即可对百度图片上的图片进行爬取
node ImageCrawler.js

```
